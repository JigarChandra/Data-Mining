Executing the program
---------------------
K-MEANS:
Run K-means as follows with parameters:
python kmeans.py <0=euclidean dist, 1= manhattan dist> <#clusters>

eg. python kmeans.py 0 16

Sample Output:
python kmeans.py 1 64
Time to cluster: 43.264
Entropy: 2.701
Std dev is 421.299


HIERARCHICAL:
Run Hierarchical as follows with parameters:
python kmeans.py <0=euclidean dist, 1= manhattan dist> <#level>

eg. python test_heir.py 0 30

Sample Output:
python test_heir.py 0 5
Entropy:3.27
number of clusters:20
